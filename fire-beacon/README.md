# Fire Beacon  
## 화재 감지 및 실시간 경고 시스템

*(IoT · 영상 인식 · 대피 경로 안내 기반 스마트 화재 대응 시스템)*

---

## 🔍 프로젝트 개요
Fire Beacon 프로젝트는 **IoT 센서 기술과 영상 인식 기술을 융합**하여  
화재 발생 상황을 조기에 감지하고, 사용자에게 **실시간 경고 및 대피 경로 안내를 제공하는  
스마트 화재 대응 시스템**을 구현한 졸업작품 프로젝트입니다.

기존 화재 감지 시스템은 단일 센서 기반으로 오탐 가능성이 존재하고,  
화재 발생 이후 사용자가 **어디로 대피해야 하는지에 대한 정보 제공이 부족**하다는 한계가 있습니다.  
본 프로젝트는 이러한 문제를 해결하기 위해  
**화재 감지 + 상황 인식 + 대피 경로 안내**를 하나의 시스템으로 통합하여 설계하였습니다.

---

## 🎯 프로젝트 목적
- 화재 발생 상황의 조기 감지를 통한 인명 및 재산 피해 최소화
- 단일 센서 기반 화재 감지 시스템의 한계 극복
- 화재 발생 시 사용자의 신속한 판단을 돕는 실시간 경고 제공
- 대피 경로 안내를 통한 안전한 피난 유도 시스템 설계
- IoT·영상 인식 기술을 결합한 통합 안전 시스템 구현 경험

---

## 🧩 시스템 구성
- **센서 기반 화재 감지**
  - 가스 센서, 온도 센서, 불꽃 감지 센서를 활용한 화재 징후 감지
- **영상 인식 시스템**
  - Raspberry Pi + 카메라를 이용한 실시간 영상 수집
  - YOLO 기반 사람(얼굴) 인식 기능 구현
- **통합 판단 구조**
  - 센서 데이터와 영상 데이터를 종합적으로 분석하여 화재 상황 판단
- **대피 경로 안내 시스템**
  - 화재 발생 위치를 고려한 안전 대피 방향 안내 구조 설계
  - 모바일 연동을 통한 대피 경로 시각화 확장 가능성 고려

---

## 🧠 주요 기능
- 가스·온도·불꽃 센서를 이용한 화재 감지
- Raspberry Pi + YOLO 기반 사람(얼굴) 인식
- 화재 감지 시 이미지 자동 캡처 및 기록
- Firebase Realtime Database를 통한 감지 데이터 저장
- Android(Kotlin) 기반 모바일 연동 구조 설계
- 화재 상황 발생 시 **대피 경로 안내를 위한 시스템 구조 설계**

---

## 📈 구현 및 실험 내용
- 센서 임계값 설정을 통한 화재 감지 조건 정의
- 영상 기반 사람 인식 기능 구현 및 동작 검증
- 화재 감지 시 이미지 캡처 및 데이터베이스 저장 기능 테스트
- 센서 단독 감지 대비 **복합 감지 구조의 안정성 및 신뢰성 확인**
- 대피 경로 안내 시스템을 고려한 확장형 구조 설계

---

## 🧩 사용 기술
- Arduino / ESP32
- Raspberry Pi + Camera Module
- Python
- YOLO (객체 인식)
- Firebase Realtime Database
- Android (Kotlin) 연동 구조 설계
