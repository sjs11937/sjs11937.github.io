# Fire Beacon Project
*(IoT · 영상 인식 기반 화재 감지 시스템)*

## 🔍 프로젝트 개요
Fire Beacon 프로젝트는 **IoT 센서 기술과 영상 인식 기술을 결합**하여  
화재 발생 상황을 조기에 감지하고 사용자에게 **실시간 경고를 제공하는  
스마트 화재 감지 시스템**을 구현한 졸업작품 프로젝트입니다.

기존 화재 감지 시스템은 단일 센서 기반으로 오탐 가능성이 존재하며,  
실시간 원격 모니터링에 한계가 있어 이를 보완하고자  
**센서 데이터 + 영상 인식 데이터를 종합적으로 분석**하는 구조로 설계하였습니다.

---

## 🎯 프로젝트 목적
- 화재 발생 상황의 조기 감지를 통한 피해 최소화
- 단일 센서 기반 시스템의 한계 극복
- IoT와 영상 인식 기술의 융합 시스템 설계 경험
- 실시간 데이터 수집·저장·모니터링 구조 구현

---

## 🧩 시스템 구성
- **센서 기반 감지 시스템**
  - 가스 센서, 온도 센서, 불꽃 감지 센서를 통한 화재 징후 감지
- **영상 인식 시스템**
  - Raspberry Pi + 카메라를 활용한 영상 수집
  - YOLO 기반 사람(얼굴) 인식 기능 구현
- **통합 분석 구조**
  - 센서 데이터와 영상 데이터를 함께 활용하여 화재 상황 판단

---

## 🧠 주요 기능
- 가스·온도·불꽃 센서를 이용한 화재 감지
- Raspberry Pi + YOLO 기반 사람(얼굴) 인식
- 화재 감지 시 이미지 자동 캡처 및 기록
- Firebase Realtime Database를 통한 감지 데이터 저장
- Android(Kotlin) 기반 모바일 연동을 고려한 시스템 구조 설계

---

## 📈 구현 및 실험 내용
- 센서 임계값 설정을 통한 화재 감지 조건 정의
- 영상 기반 사람 인식 기능 구현 및 테스트
- 화재 감지 시 이미지 저장 및 데이터베이스 기록 동작 검증
- 센서 단독 감지 대비 **복합 감지 구조의 안정성 확인**

---

## 🧩 사용 기술
- Arduino / ESP32
- Raspberry Pi + Camera Module
- Python
- YOLO (객체 인식)
- Firebase Realtime Database
- Android (Kotlin) 연동 구조 설계
