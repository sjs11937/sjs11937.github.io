# Fire Beacon Project

## 🔥 프로젝트 개요
Fire Beacon은 IoT 및 영상 인식 기술을 활용하여 화재 발생 상황을 조기에 감지하고 사용자에게 경고를 제공하는 스마트 화재 감지 시스템 졸업작품 프로젝트입니다.

ESP32 기반의 센서 모듈과 Raspberry Pi + 카메라를 활용하여 가스, 온도, 불꽃(영상) 정보를 종합적으로 분석하도록 설계되었습니다.

---

## 🧠 주요 기능
- 가스감지, 온도감지, 불꽃감지 센서를 이용한 화재 감지
- Raspberry Pi + YOLO 기반 사람(얼굴) 인식
- 화재 감지 시 이미지 캡처 및 기록
- Firebase Realtime Database를 통한 데이터 저장
- Android(Kotlin) 기반 연동 가능 구조 설계
